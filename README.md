# TodoList

## Before you start

Make sure you have [Node.js v20](https://nodejs.org/en/) installed.

Run the following command to install dependencies:

```sh
npm install
```

## Serving Database Mock

```sh
npx json-server db.json
```

## Serving Application

```sh
npm run start
```

## Building Application

```sh
npm run build
```

## Linting

```sh
npm run lint
```

## Stylelint

```sh
npm run stylelint
```

## Testing

```sh
npm run test
```
